Projects Overview

Welcome to my Projects portfolio! This page showcases a collection of my key programming and web development projects, demonstrating hands-on experience with various technologies and problem-solving skills. Each project includes a brief description, the tools and languages used. Feel free to explore and learn more about my work.

Project 1 : AI Video Summarization and Quiz Generation

A tool that transcribes educational videos to text using speech recognition, then summarizes the content and generates quiz questions automatically. Technologies include Large Language Models (LLMs) and speech-to-text models like Whisper. Helps automate learning material creation.

Project 2 : Stock Price Prediction

A stock predictor uses advanced algorithms, historical data, and other indicators to forecast the future value and movements of stocks or other financial instruments. These predictions, which aim to provide insights for trading and investment, are not guaranteed to be accurate due to the market's inherent complexity and unpredictability.

Project 3 : Credit Card Fraud Detection

A credit card fraud predictor is an advanced system, primarily powered by artificial intelligence (AI) and machine learning (ML), used by financial institutions to analyze transaction data in real-time and identify suspicious activities that may indicate fraud. Instead of relying solely on static, predefined rules, modern fraud predictors leverage vast amounts of historical transaction data to learn and adapt to emerging fraud patterns.
